# IAChatbot - Agente Inteligente Automatizado

Este proyecto consiste en un agente de IA desarrollado en Python, diseñado para integrarse con flujos de trabajo en **n8n** para automatizar tareas repetitivas.

### Visión del Proyecto
Crear un asistente capaz de entender intenciones del usuario y disparar acciones automáticas (como correos, registros en bases de datos o alertas) mediante webhooks.

### Componentes Técnicos
- **Lenguaje:** Python
- **Lógica de IA:** Clasificador de intenciones (NLP).
- **Integración:** n8n (Workflows asíncronos).
